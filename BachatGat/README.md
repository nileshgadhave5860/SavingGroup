# BachatGatAPI Workspace

Projects:
- BachatGatAPI (Web API)
- BachatGatDAL (Class Library)
- BachatGatBAL (Class Library)
- BachatGatDTO (Class Library)

Quick start (requires .NET 8 SDK):

```bash
dotnet --version
dotnet build BachatGatAPI/BachatGatAPI.csproj
dotnet run --project BachatGatAPI/BachatGatAPI.csproj
```

API will run and Swagger UI available in development at `/swagger`.
